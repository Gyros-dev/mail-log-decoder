# KASSIR.RU — декодер почтовых логов

Статический проект без сервера и базы данных. Подходит для размещения на GitHub Pages.

## Возможности

- декодирование MIME-фрагментов `=?charset?Q?...?=` и `=?charset?B?...?=` в читаемый текст;
- поддержка Q- и Base64-кодирования, склейка разбитых на части заголовков;
- раскрытие CSV-экранирования (`""`, `\n`, `\t`);
- автоматическое декодирование при вставке;
- копирование результата в буфер обмена;
- всё работает локально в браузере, данные никуда не отправляются.

## Запуск локально

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## GitHub Pages

1. Создайте репозиторий.
2. Загрузите все файлы проекта в ветку `main`.
3. Откройте **Settings → Pages**.
4. Выберите **Deploy from a branch**, ветку `main`, папку `/ (root)`.
